## 🚀 部署到Vercel

1. **清空GitHub仓库**
2. **上传这些文件：**
   - `index.html`
   - `api/subscribe.js`

3. **在Vercel中：**
   - Framework Preset: **Other**
   - Root Directory: `./`
   - 点击 Deploy

4. **添加域名：**
   - Settings → Domains
   - 添加 `labubuwallpaper.shop`
